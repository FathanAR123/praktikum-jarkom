# Laporan Praktikum Jarkom IF
## Langkah-langkah Percobaanya 
1. Buka aplikasi Wireshark
Pertama, buka aplikasi Wireshark dan browser. Setelah itu, klik kanan pada jaringan Wi-Fi lalu pilih Start Capture untuk mulai menangkap seluruh paket jaringan.

2. Memasuki proses capture jaringan 
Ada banyak sekali paket jaringan yang ditangkap oleh komputer kita tetapi disini masih belum pasti yang mana aktifitas yang browser kita lalkukan

3. Membuat dan memperoleh aktivitas jaringan pada browser
Selanjutnya kita akan membuat dan memperoleh aktifitas jaringan menggunakan browser dengan metode link yang disediakan.
(http://gaia.cs.umass.edu/wireshark-labs/INTRO-wireshark-file1.html)

4. Menghentikan penankapan jaringan capture
Setelah membuka website sebelumnya ktia akan menghentikan agar menampilkan daftar paket apa saja yang sudah direkam selama kita membuat link website sebelumnya

5. Memfilter paket jaringan
Disini kita akan mencari paket IP kita secara spesifik dengan menggunakan filter lalu ketik “http”. Disitu kita bisa melihat ada  paket IP utama kita yang dimana IP tersebut dihasilkan dari web yang baru saja kita buka tadi
